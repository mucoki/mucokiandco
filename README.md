## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #d4af37;    /* Gold */
    --dark-color: #1a1a1a;       /* Dark */
    --light-color: #f5f5f5;      /* Light */
    --accent-color: #8b7500;     /* Accent */
}
```

### Contact Information
Update contact details in:
- `contact.html` - Contact information section
- `index.html` - Footer
- All pages - Footer contact

### Products
Edit product information directly in `catalog.html`
